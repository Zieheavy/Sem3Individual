## About The Project
This project is basicly the orignal pong arcade game. That has been turned into an online version where 2 players can play against each other. You can play the game via the webbroswer or via an desktop application. 


### Built With

* [Vue.js](https://vuejs.org/)
* [Unity](https://docs.unity3d.com/ScriptReference/)
* [ASP.Net](https://docs.microsoft.com/en-us/aspnet/core/?WT.mc_id=dotnet-35129-website&view=aspnetcore-6.0)
